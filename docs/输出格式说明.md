# 输出格式说明

`scripts/run_clean.py` 会生成三类 CSV 和一份统计报告：

- `auto_mapped.csv`
- `need_review.csv`
- `manual_required.csv`
- `stats_report.txt`

CSV 使用 `utf-8-sig` 编码，便于 Excel 打开中文。

## 字段说明

### P0 原始上下文

| 字段 | 说明 |
|---|---|
| `study_id` | 体检记录 ID |
| `exam_time` | 体检时间 |
| `package_name` | 套餐名称 |
| `dept_code` | 科室编码 |
| `dept_name` | 科室名称 |
| `source_table` | 来源表 |
| `major_item_code` | 大项编码 |
| `major_item_name` | 原始大项名称 |
| `item_code` | 原始指标编码 |
| `item_name` | 原始指标名称 |
| `item_name_en` | 原始英文缩写 |

### P0 解析结果

| 字段 | 说明 |
|---|---|
| `result_value_raw` | 原始结果值 |
| `numeric_value` | 解析后的数值结果 |
| `text_value` | 解析后的文本结果 |
| `unit` | 标准化后的最终单位 |
| `unit_raw` | 原始单位字段 |
| `unit_parsed` | 从结果值中解析出的单位 |
| `qualifier` | `<`、`>` 等修饰符 |
| `judgment` | `偏低`、`正常`、`偏高`、`异常` |
| `ref_in_value` | 从结果值中拆出的参考范围 |
| `reference_range_raw` | 原始参考范围 |
| `ref_min` | 结构化参考范围下限 |
| `ref_max` | 结构化参考范围上限 |
| `ref_text` | 文本型参考范围 |
| `ref_conditions` | 多条件参考范围 |
| `is_simple_range` | 是否为简单数值范围 |
| `abnormal_flag` | HIS 原始异常标记 |
| `abnormal_flag_source` | 异常标记来源值 |
| `is_abnormal` | 综合异常判定 |
| `abnormal_direction` | `high`、`low` 或空 |
| `major_item_standard_code` | 标准化后的大项编码 |
| `major_item_standard_name` | 标准化后的大项名称 |
| `major_item_category` | 标准化后的大项分类 |

### 标准化结果

| 字段 | 说明 |
|---|---|
| `original_name` | 进入标准化流程的原始名称 |
| `cleaned_name` | L1 清洗后的名称 |
| `abbreviation` | 提取出的英文缩写 |
| `standard_name` | 匹配到的标准名称 |
| `standard_code` | 标准指标编码 |
| `category` | 标准分类 |
| `standard_unit` | 标准字典中的常见单位 |
| `result_type` | 结果类型，如 numeric、qualitative |
| `confidence` | 当前映射置信度 |
| `match_source` | `alias_exact`、`abbr_exact`、`l2_embedding`、`ai_review`、`ai_review_uncertain` 或 `unmatched` |
| `top_candidates` | L2 候选列表 |
| `ai_review` | AI 复核结果摘要 |

### 审核候选字段

| 文件 | 字段 | 说明 |
|---|---|---|
| `need_review.csv` | `top3_candidates` | 前 3 个候选，JSON 字符串 |
| `manual_required.csv` | `top5_candidates` | 前 5 个候选，JSON 字符串 |

## 分层规则

| 置信度 | 输出文件 | 说明 |
|---|---|---|
| `>= 0.95` | `auto_mapped.csv` | 自动归一 |
| `0.80 <= confidence < 0.95` | `need_review.csv` | 待审核 |
| `< 0.80` | `manual_required.csv` | 人工处理 |

启用 AI 复核后，AI 明确判断的记录会以 `match_source=ai_review` 进入自动归一；AI 仍不确定的记录会以 `match_source=ai_review_uncertain` 留给人工处理。

- `src/indicator_aggregator.py`：生成 HPV / 食物不耐受摘要
- `src/risk_analyzer.py`：计算偏离度并分类四象限
