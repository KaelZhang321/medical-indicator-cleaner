# 前端数据接口方案

## 技术栈

- **FastAPI** — REST API 服务
- **Uvicorn** — ASGI 服务器
- 复用现有所有 src/ 模块，不重复造轮子

## API 接口设计

### 1. 客户信息查询

#### GET `/api/v1/exam/{study_id}`

查询单次体检的标准化结果。

**响应**：
```json
{
  "study_id": "2512125012",
  "patient_name": "李某某",
  "gender": "女",
  "exam_time": "2025-12-12",
  "package_name": "海-033VIPGB女-丽滋高致精筛套餐",
  "summary": {
    "total_indicators": 173,
    "abnormal_count": 6,
    "categories": ["血脂", "肝功能", "肾功能", ...]
  },
  "indicators": [
    {
      "standard_code": "040201",
      "standard_name": "总胆固醇",
      "category": "血脂",
      "value": 5.65,
      "unit": "mmol/L",
      "reference_range": "0-5.2",
      "is_abnormal": true,
      "abnormal_direction": "high"
    },
    ...
  ]
}
```

#### GET `/api/v1/patient/{sfzh}/exams`

查询某患者所有体检记录列表。

**响应**：
```json
{
  "patient_id": "****",
  "exam_count": 3,
  "exams": [
    {"study_id": "2512125012", "exam_time": "2025-12-12", "package_name": "...", "abnormal_count": 6},
    {"study_id": "2406150088", "exam_time": "2024-06-15", "package_name": "...", "abnormal_count": 3},
  ]
}
```

### 2. 指标纵向对比

#### GET `/api/v1/patient/{sfzh}/comparison`

查询某患者多次体检的指标纵向对比。

**Query 参数**：
- `category`（可选）：过滤特定分类，如 `血脂`

**响应**：
```json
{
  "patient_id": "****",
  "exam_dates": ["2024-06-15", "2025-12-12"],
  "comparisons": [
    {
      "standard_code": "040201",
      "standard_name": "总胆固醇",
      "category": "血脂",
      "unit": "mmol/L",
      "values": {"2024-06-15": 4.8, "2025-12-12": 5.65},
      "trend": "↑",
      "ref_min": 0.0,
      "ref_max": 5.2
    },
    ...
  ]
}
```

### 3. 四象限分析

#### GET `/api/v1/exam/{study_id}/quadrant`

对单次体检做四象限分析。

**响应**：
```json
{
  "study_id": "2512125012",
  "quadrants": {
    "紧急处理": [
      {"standard_name": "总胆固醇", "deviation": 1.35, "risk_weight": 0.6, "value": 5.65, "ref_max": 5.2}
    ],
    "重点关注": [...],
    "轻度异常": [...],
    "正常范围": [...]
  },
  "stats": {
    "urgent_count": 1,
    "watch_count": 2,
    "mild_count": 3,
    "normal_count": 80
  }
}
```

### 4. 疗效预测特征

#### GET `/api/v1/patient/{sfzh}/features`

生成患者的疗效预测基础特征。

**响应**：
```json
{
  "patient_id": "****",
  "exam_count": 2,
  "features": {
    "abnormal_count": 6,
    "040201_latest": 5.65,
    "040201_change_rate": 0.177,
    ...
  }
}
```

## 项目结构

```
api/
├── __init__.py
├── main.py              # FastAPI 应用入口
├── routers/
│   ├── __init__.py
│   ├── exam.py          # /exam 路由
│   ├── patient.py       # /patient 路由
│   └── analysis.py      # /quadrant, /features 路由
├── deps.py              # 依赖注入（DB连接、Pipeline等）
└── schemas.py           # Pydantic 响应模型
```

## 启动方式

```bash
DB_PASSWORD='xxx' ARK_API_KEY='xxx' uvicorn api.main:app --host 0.0.0.0 --port 8000
```

自动文档：`http://localhost:8000/docs`
