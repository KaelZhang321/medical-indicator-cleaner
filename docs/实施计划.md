### M3-2 编写 `src/result_parser.py` — parse 主方法
- **文件**: `src/result_parser.py`（追加）
- **需实现**:
  - `parse(self, raw_value: str, existing_unit: str = '') -> dict`
  - 返回格式:
    ```python
    {
        'numeric_value': float | None,
        'text_value': str | None,
        'unit': str,
        'qualifier': str | None,   # '<', '>'
        'judgment': str | None,     # '偏低', '正常', '偏高'
        'ref_in_value': str | None, # 从值中提取的参考范围
    }
    ```
  - 处理逻辑（按优先级）:
    1. 如果 raw_value 为 None 或空 → 返回全空结果
    2. 如果 existing_unit 非空 → 直接使用，不从值中提取单位
    3. 提取修饰符: `< 2.30` → qualifier='<', 剩余='2.30'
    4. 提取单位: `28.2kg` → unit='kg', 剩余='28.2'
    5. 提取括号内参考范围: `(7.7-9.4)` → ref_in_value='7.7-9.4'
    6. 提取判断词: `偏低`/`正常`/`偏高` → judgment
    7. 提取逗号分隔: `80.8,正常` → numeric=80.8, judgment=正常
    8. 提取定性描述: `0.07（阴性 -）` → numeric=0.07, text=阴性
    9. 纯文本: `O型`/`清亮`/`阳性（+）` → text_value
    10. 纯数值: `1.74` → numeric_value=1.74
- **验收**: 所有 M3-3 测试通过
